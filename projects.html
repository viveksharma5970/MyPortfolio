LeetMetric – LeetCode Progress Tracker

LeetMetric is a sleek web application designed to track and visualize a user's LeetCode problem-solving performance in real time. It’s a productivity tool for coders aiming to stay on top of their DSA journey.

  • Enter any LeetCode username to fetch live stats
  • Visualize Easy, Medium, and Hard question progress via animated circular charts
  • View total submissions for each difficulty level
  • Validates usernames using regex to ensure proper format
  • Fully responsive design with clean UI
  • Built using HTML, CSS, JavaScript and an open LeetCode API

Library Management System – Console-Based Application

This is a simple and efficient C++ console application designed to manage a small-scale library. It handles book records, issues, and returns with ease using Object-Oriented Programming principles.

  • Add new books with unique ID and title validation
  • Search for books using title or ID
  • Issue books to students by storing their name and registration number
  • Return books and update availability status
  • Delete books from the library collection
  • View all available and issued books with detailed information
  • Developed using C++ with OOPs concepts like classes and structures
